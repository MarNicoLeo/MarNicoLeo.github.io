2017-08-02

工作日报

日报总结

第一天

  • 上午完成空调温度推送，报警推送；温湿度传感器温度推送，云图呈现；漏水线摆放并推送警告；阅读可视化配置用户手册。

  • 下午完成涛哥布置综合案例的以下任务（明天上午继续）：

    1. 创建并导入场景；
    2. 导入cad文件到visio中，并调整比例；
    3. 以组的方式在visio中摆放机柜、配电列头柜、空调等（结合照片和手绘草图，模拟真实场景），并将布局导入系统，完成基本效果展示；
    4. 编辑台账，插入了一部分架式设备和板卡

  • 现在看来任务不是很难，主要是细节多，还有一些概念不了解。

    注意总结，勤练习都能应对。

第二天

  • 上午完成了通风地板、天花板、温湿度传感器的摆放，漏水线的绘制，并通过实施工具向温湿度感应器推送温度，向漏水线和空调推送警告。
  • 下午解决给空调推送温度，点击动环中的空调按钮不显示温度标签的问题（原因是模型库中的这个空调模型的自定义标签不是空调）。另外整理用户的台账表，提炼出自己需要的台账表并导入台账。
  • 今天的分享会很有收获，之前也用过gulp，不过仅仅使用了压缩、合并、重命名等功能；另外DCV组的几位前端同事看起来都很强，期待加入他们一起进步。

第三天

  • 上午首先完善前一天的工作，比如导入了配线，并修改配线颜色等其他工作。
  • 下午主要是接到绘制云南高检办公区二层的Visio绘制任务，听师姐讲解绘制要求，自己分析cad图纸，后来收到相关图纸的照片实拍，使得看图更容易一些。实际工作完成了业务楼二层的房间绘制（吐槽Visio老是提示不闭合，不是那么智能）并摆放了部分设备。
  • 实施组的同事们也是相当辛苦的，光是Visio布局就挺繁琐的。但是我也要争取保质保量完成任务。

第四天

  • 上午完成云南高检办公区二层的设备摆放，并开始做办公区三层绘制工作。
  • 下去完成晓丹姐布置的三个相关任务，目的在于验证可视化配置文档的易用性；完成办公区三层房间绘制，并摆放部分设备。
  • 云南高检这个项目看来是挺大，小武已经在云南待了20几天了。抓紧干吧，第一个任务我一定要干好。

第五天

  • 上午完成办公区三层设备摆放，并开始综合楼三层房间绘制。
  • 下午完成综合楼三层房间及设备摆放工作，并发送给小武。
  • 我们DCV组的团队很棒，期待尽快加入。

第六天

  • 上午完成业务楼三层房间绘制和设备摆放。
  • 下午完成综合楼四层房间绘制和设备摆放，期间图纸有问题，及时同小武进行交流，最后将完成版本发送给小武。
  • 快下班时开始安装眼镜猴，比起画图多了很多乐趣，可以互相讨论，这不仅增进了同事间的感情，也让我了解了一些有用的知识。

第七天

  • 上午完成综合楼五层大部分工作
  • 下午完成上午剩余的工作，并完成眼镜猴部署。
  • 今天收获挺大！

第八天

  • 上午安装了centOS版本的Tarsier，期间遇到了一些问题，我自己研究安装文档后，重新解压重新装了两遍才解决。
  • 下午开始以最小化安装的方式安装centOS版本，也遇到很多问题，最后卡在2002的错误上，由于开会，没请到大神来解决问题，下周回去尽快搞定。
  • 今天小赵同学也开始做涛哥布置的那个比较复杂的案例，今天过去他那边比较多。

第九天

  • 上午完成上周余下部分任务，并听取廖同学讲解IT架构，之后开始自己练习。
  • 下午安装古老版本的ITVOP（一个看起来比较全面和复杂demo），熟悉里面的组成和功能，根据涛哥要求，明天我要在眼镜猴上实现与demo完全相同的功能。
  • 新的一周加油！

第十天

  • 今天涛哥没过来，上午首先接着熟悉老版本的ITVOP，之后开始查找页面切换特效。
  • 下午依然是在查找，找到一个特效后开始查看他的源码。这个案例的代码相对简单一些，实现起来不是很复杂，当做保底的。后面也找到两个，明天空闲时间再对比查找一下。
  • 眼镜猴页面要比老版本的好看很多，赶明导进去数据抓紧熟悉熟悉。

第十一天

  • 上午主要是在眼镜猴上部署一个demo，从导入场景一直到导入配线的过程大体和旧版本一致，不同点是台账和配线被放到了配置里面
  • 下午开始在模模搭中给机柜编号，总共有七层楼，已经完成四层
  • 无

第十二天

  • 上午主要完成前一天剩余的几个楼层的机柜编号，期间将页面切换效果实现了一版。
  • 下午接到任务是将ITVOP上的ITV搬到DMV上去，我自己尝试了好一会儿，没能成功。之后请教同事以及涛哥得知，ITVOP导出来的数据没法直接导入到DMV，需要重新画图。未来一周的重大任务就是重画这个几十张拓扑图。

第十三天

  • 上午开始画图，但涛哥给的数据拖入到眼镜猴上图标不显示，涛哥上午很忙，也带着其他实施组的同事一起开会。一开始我想在那台没人用的台式机上部署这个老DCV的demo，最后决定试着在我的笔记本上同时开启眼镜猴和老demo，果然电脑卡的不行，但是在老demo上图标能显示出来，也能在眼镜猴上进行画图。
  • 下午，因为我之前对DMV的逻辑一直涉及不多，所以请其他同事帮忙讲了讲，最后自己画了三张图。
