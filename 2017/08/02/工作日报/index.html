工作日报

日报总结

第一天

  • 上午完成空调温度推送，报警推送；温湿度传感器温度推送，云图呈现；漏水线摆放并推送警告；阅读可视化配置用户手册。

  • 下午完成涛哥布置综合案例的以下任务（明天上午继续）：

    1. 创建并导入场景；
    2. 导入cad文件到visio中，并调整比例；
    3. 以组的方式在visio中摆放机柜、配电列头柜、空调等（结合照片和手绘草图，模拟真实场景），并将布局导入系统，完成基本效果展示；
    4. 编辑台账，插入了一部分架式设备和板卡

  • 现在看来任务不是很难，主要是细节多，还有一些概念不了解。

    注意总结，勤练习都能应对。

第二天

  • 上午完成了通风地板、天花板、温湿度传感器的摆放，漏水线的绘制，并通过实施工具向温湿度感应器推送温度，向漏水线和空调推送警告。
  • 下午解决给空调推送温度，点击动环中的空调按钮不显示温度标签的问题（原因是模型库中的这个空调模型的自定义标签不是空调）。另外整理用户的台账表，提炼出自己需要的台账表并导入台账。
  • 今天的分享会很有收获，之前也用过gulp，不过仅仅使用了压缩、合并、重命名等功能；另外DCV组的几位前端同事看起来都很强，期待加入他们一起进步。

第三天

  • 上午首先完善前一天的工作，比如导入了配线，并修改配线颜色等其他工作。
  • 下午主要是接到绘制云南高检办公区二层的Visio绘制任务，听师姐讲解绘制要求，自己分析cad图纸，后来收到相关图纸的照片实拍，使得看图更容易一些。实际工作完成了业务楼二层的房间绘制（吐槽Visio老是提示不闭合，不是那么智能）并摆放了部分设备。
  • 实施组的同事们也是相当辛苦的，光是Visio布局就挺繁琐的。但是我也要争取保质保量完成任务。

第四天

  • 上午完成云南高检办公区二层的设备摆放，并开始做办公区三层绘制工作。
  • 下去完成晓丹姐布置的三个相关任务，目的在于验证可视化配置文档的易用性；完成办公区三层房间绘制，并摆放部分设备。
  • 云南高检这个项目看来是挺大，小武已经在云南待了20几天了。抓紧干吧，第一个任务我一定要干好。

第五天

  • 上午完成办公区三层设备摆放，并开始综合楼三层房间绘制。
  • 下午完成综合楼三层房间及设备摆放工作，并发送给小武。
  • 我们DCV组的团队很棒，期待尽快加入。
